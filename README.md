# To-Do List App (Python)

This is a GUI-based To-Do List application built using Python and Tkinter.

## Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Save tasks using file storage

## Technologies Used
- Python
- Tkinter

## How to Run
1. Install Python
2. Run the file:
   python todo_gui.py
